Firecrawl API (2.0.0)

Download OpenAPI specification:

E-mail: support@firecrawl.com URL: https://firecrawl.dev

Firecrawl API allows you to turn any website into clean markdown or structured data. With Firecrawl, you can scrape web pages, crawl entire websites, and extract structured information.

Authentication

All requests require an API key in the Authorization header:

Authorization: Bearer fc-YOUR_API_KEY

Base URL

All API requests are made to:

https://api.firecrawl.dev

Rate Limiting

The API enforces rate limits to ensure fair usage. Exceeding the rate limit returns a 429 status code.

Core Operations

Core scraping, crawling, mapping, and search operations
